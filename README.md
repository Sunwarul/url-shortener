# URL Shortener App

## Installation:

### Requirements:

- PHP 8.2+
- MySQL
- Composer (For PHP Dependency)
- Node (Npm)

### Dependency install:

- `composer install`
- `npm install && npm run build`
- `php artisan migrate`
- `php artisan db:seed`

### To run the application locally:

- `php artisan serve`
- Open `http://localhost:8000` in your browser to access
